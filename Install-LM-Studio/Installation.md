LM Studio provides a desktop interface for running large language models (LLMs) locally with GPU acceleration. On Linux, the easiest way to install it is via the **AppImage** package.

---

## 1. Download LM Studio AppImage
Visit the official LM Studio site:

ðŸ‘‰ [LM Studio Downloads](https://lmstudio.ai/home)

Download the latest `.AppImage` release for Linux (e.g., `LM_Studio-x86_64.AppImage`).

---

## 2. Make the AppImage Executable
Open a terminal in the directory where the file was downloaded and run:

```bash
chmod +x LM_Studio-x86_64.AppImage
```

Start LM Studio directly or by clicking on the appimage icon
```bash
./LM_Studio-x86_64.AppImage
```
